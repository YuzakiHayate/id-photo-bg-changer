## 一、产品概述

### 1.1 产品定位
一个轻量级在线工具，帮助用户快速将证件照更换背景颜色，支持常见证件照规格，满足签证、简历、报名等场景需求。

### 1.2 目标用户
- 需要制作/修改证件照的个人用户（求职、考试报名、签证申请等）
- 对 PS 操作不熟悉、希望快速处理的普通用户

### 1.3 核心价值
上传一张照片，10 秒内换好底色，下载即用。

## 二、MVP 范围

### 2.1 核心功能（必须有）

| # | 功能 | 说明 |
|---|------|------|
| F1 | 图片上传 | 支持 JPG/PNG，最大 10MB |
| F2 | 自动抠图 | 调用 remove.bg API 去除背景 |
| F3 | 背景换色 | 提供红、蓝、白三种预设色 + 自定义颜色选择器 |
| F4 | 实时预览 | 换色后即时预览效果 |
| F5 | 图片下载 | 下载换色后的 JPG 图片 |
| F6 | 免费额度 | 每个用户每天免费处理 2 张 |
| F7 | 付费解锁 | 超出免费额度后引导付费 |

### 2.2 不在 MVP 范围内（后续迭代）
- 用户注册/登录系统（MVP 用本地存储记录免费次数）
- 证件照裁剪/规格调整
- 批量处理
- 人脸居中检测
- 历史记录
- 移动端 App

## 三、用户流程
访问网站 → 上传照片（拖拽或点击）→ 自动调用 remove.bg 抠图 → 展示抠图结果 + 背景色选择器 → 用户选择目标背景色（实时预览）→ 免费次数未用完则直接下载，已用完则弹出付费引导弹窗 → 下载完成

## 六、技术方案

### 6.1 技术栈

| 层 | 技术 |
|----|------|
| 前端框架 | Next.js 14 (App Router) |
| 样式 | Tailwind CSS |
| 图像合成 | 浏览器 Canvas API |
| 抠图 API | remove.bg API |
| API 代理 | Cloudflare Workers |
| 部署 | Cloudflare Pages |
| 免费次数限制 | localStorage（MVP 阶段）|

### 6.2 系统架构
用户浏览器 → 上传图片 → Cloudflare Worker (/api/remove-bg) → 转发到 remove.bg API → 返回透明背景 PNG → Canvas 合成（填充背景色 + 叠加人像）→ 下载最终图片

### 6.3 关键接口
**POST /api/remove-bg**
- 请求：multipart/form-data，包含图片文件
- 响应：透明背景的 PNG 图片（binary）
- 作用：代理 remove.bg API，隐藏 API Key

### 6.4 remove.bg API 说明
- 文档：https://www.remove.bg/api
- 免费额度：50 次/月（开发测试用）
- 收费：约 $0.2/张（按量付费）
- 响应时间：通常 1-3 秒

## 十、风险与注意事项
- **remove.bg 费用：** 按量计费，需要监控调用量，避免超支。MVP 阶段建议设置每日调用上限。
- **免费次数绕过：** localStorage 可被用户清除绕过，MVP 阶段可接受，后续需要服务端验证。
- **图片隐私：** 明确告知用户图片不会被存储，增加用户信任。
- **国内访问 remove.bg：** remove.bg 在国内访问可能不稳定，Cloudflare Worker 作为代理可以缓解，但需测试。
